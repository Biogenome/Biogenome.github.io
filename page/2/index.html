模块

为了编写可维护的代码，我们把很多函数分组，分别放到不同的文件里，这样，每个文件包含的代码就相对较少，很多编程语言都采用这种组织代码的方式。在Python中，一个.py文件就称之为一个模块（Module）.Python的模块其实就是封装了一个或者多个功能的代码集合，以便于重用，模块可以是一个文件也可以是一个目录，目录的形式称作包。使用模块的好处：

  • 大大提高了代码的可维护性。
  • 编写代码不必从零开始。
  • 使用模块还可以避免函数名和变量名冲突。
  • 模块名字与其他模块冲突了，于是我们可以通过包来组织模块，避免冲突，自己创建模块时要注意命名，不能和Python自带的模块名称冲突。

git简介

Git是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。分布式版本控制系统除了Git以及促使Git诞生的BitKeeper外，还有类似Git的Mercurial和Bazaar等。这些分布式版本控制系统各有特点，但最快、最简单也最流行的依然是Git！

  • 相比于集中式版本控制系统，分布式版本控制系统根本没有“中央服务器”，每个人的电脑上都是一个完整的版本库。

  • 分布式版本控制系统的安全性更高。

  • 免费,最快，最简单。

爬虫基本结构

正常通过浏览器可以获取的数据，爬虫都可以获取,爬虫的本质是一个服务端，实现的功能类似于浏览器.是一种自动获取网页内容的程序。是搜索引擎的重要组成部分.我们需要干的事情其实就两件事情：找到我们需要的网页，然后把他们一个一个处理一遍。所以，爬虫的核心模块有三个：请求、解析、储存

  • 对于单个页面来说，入口的来源有两种，一种是已知地址,另外一种是你可以通过前面爬下来的网页中获取入口.例如浏览网页的过程，用户输入网址之后，经过 DNS 服务器，找到服务器主机，向服务器发出一个请求，服务器经过解析之后，发送给用户的浏览器 HTML、JS、CSS 等文件，浏览器解析出来，用户看到的网页实质是由 HTML 代码构成的，通过分析和过滤这些 HTML 代码，实现资源的获取。
