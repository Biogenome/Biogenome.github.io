第2天-学习python

发表于 2017-04-15   |   分类于 技术

python安装

从官网上下载python安装程序,按照自己电脑版本(64位或32位)，下载相应的安装包，可以选择默认安装或自定义安装盘。

  1. 在安装过程中记得勾选add to path选项，将python程序安装到变量路径中
  2. 安装完成，在电脑命令行中运行python,如果提示python不是系统命令，可以手动添加环境变量，具体方法见win7环境变量:环境变量PATH修改方法,或者可以用cd命令修改当前目录，使程序运行。具体见python 如何运行程序

python程序运行

要运行python脚本，可以有几个方面来实现：

  • 直接在命令行里运行：在DOS窗口里，输入python就可以进入python的运行窗口，然后在里面写代码。linux或者unix系统，很多都会自动给你安装python的运行环境，你只要在shell目录下输入python就可以看到
    python交互式命令行的环境了。
  • 直接保存成文件的形式运行：你把写好的文件放到保存成.py的后缀，比如你保存写了几行python代码。保存成test.py的文件名放到到d:\pythoner。那文件的路径就是d:\pythoner\test.py。如果你要运行test.py的话，只需要在dos里，输入python d:\pythoner\test.py或者是你进入到这个目录，然后运行脚本：python test.py。
  • 选择一点的编辑器，如python自带的IDEL集成开发编辑器等。比如我用的是sublime txt,免费使用，脚本写完之后，直接按快捷键ctrl+B就可以运行，运行结果显示在下方。或者可以选择微软的VSCode文本编辑器，国内好评较高的notepad++编辑器，看你个人爱好咯！
  1. 绝对不能用Word和Windows自带的记事本。Word保存的不是纯文本文件，而记事本会自作聪明地在文件开始的地方加上几个特殊字符（UTF-8 BOM），结果会导致程序运行出现莫名其妙的错误。
  2. 必须要以.py结尾，其他的都不行。
  3. 文件名或变量名只能是英文字母、数字和下划线的组合

python语言规则

  • #开头的语句是注释，当语句以冒号:结尾时，缩进的语句视为代码块。按照约定俗成的管理，应该始终坚持使用4个空格的缩进。
  • Python程序是大小写敏感的，如果写错了大小写，程序会报错。
  • 变量名必须是大小写英文、数字和_的组合，且只能以数字开头。可以有两种命名方法，第一种为第一个单词小写，第二个单词首字母大写，第二种为每一个单词的首字母都大写，或者可以用下划线来链接连个单词。
  • 在Python中，等号=是赋值语句，可以把任意数据类型赋值给变量，同一个变量可以反复赋值，而且可以是不同类型的变量。
  • 在Python中，通常用全部大写的变量名表示常量，用全部大写的变量名表示常量只是一个习惯上的用法。

Markdown 概述

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成格式丰富的HTML页面。- 维基百科
Markdown 语法是对纯文本格式的强化，能使文本显示得更清晰、有条理。但它依旧算是简单的文本，很容易修改和扩展，常用于快速写作中。Markdown 格式的简洁特性、兼容扩展性颇佳，使之能快速转换为各种互联网上的常用格式，比如 HTML、Word、PDF 等。目前，越来越多的人开始接受和使用它。

庐山简介

庐山，又称匡山、匡庐，位于中国江西省九江市南郊,鄱阳湖西北岸,是联合国教科文组织评定的文化遗产和世界地质公园.也是国家5A级旅游景区和文化名山,自东晋以来，中国历代著名的文人、高僧、政治人物都在此留下过重要的历史印迹.西汉史学家司马迁游山后，在其著述《史记·河渠书》中有“余南登庐山，观禹疏九江”的句子，这是“庐山”之名第一次见诸于文字记录，也是最早的记录，一直沿用至今。
