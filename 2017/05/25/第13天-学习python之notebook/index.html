Jupyter Notebook 快速入门

Jupyter notebook 前身为IPython Notebook，学习时，可以找两者的教程,目前，最新版本的Anaconda是自带Jupyter NoteBook的，不需要再单独安装,因为windows不具有terminal需要的TTY，所以，windows下的jupyter是不支持Terminal模式(在notebook new新建中)，如果想新建一个 notebook，只需要点击New，选择你希望启动的 notebook 类型即可。

notebook 界面组成

  1. notebook 的名称
  2. 主工具栏，提供了保存、导出、重载 notebook，以及重启内核等选项
  3. 快捷键
  4. notebook 主要区域，包含了 notebook 的内容编辑区

下方的主要区域，由被称为单元格的部分组成。每个 notebook 由多个单元格构成，而每个单元格又可以有不同的用途。一个代码单元格（code cell），以[ ]开头。在这种类型的单元格中，可以输入任意代码并执行。可以修改之前的单元格，对其重新计算，这样就可以更新整个文档了。你也可以重新计算整个 notebook，只要点击Cell-> Run all即可。

  • 首先，我们在顶部添加一个 notebook 的标题。选中第一个单元格，然后点击Insert -> Insert单元格above（在上方插入单元格）。你会发现，文档的顶部马上就出现了一个新的单元格。点击在快捷键栏中的单元格类型，将其变成一个标题单元格（heading cell），这个单元格以#标记开头，意味着这是一个一级标题。如果需要子标题，同Markdown语法。
  • 可以重命名该 notebook，点击Fiel -> Rename，然后输入新的名称。这样，新的名称将会出现在窗口的左上角，在 Jupyter 的标志旁边。
