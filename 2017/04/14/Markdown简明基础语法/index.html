Markdown简明基础语法

发表于 2017-04-14   |   分类于 技术

Markdown 概述

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成格式丰富的HTML页面。- 维基百科
Markdown 语法是对纯文本格式的强化，能使文本显示得更清晰、有条理。但它依旧算是简单的文本，很容易修改和扩展，常用于快速写作中。Markdown 格式的简洁特性、兼容扩展性颇佳，使之能快速转换为各种互联网上的常用格式，比如 HTML、Word、PDF 等。目前，越来越多的人开始接受和使用它。

Markdown 编辑器

  • 常见编辑器: MacDown,Typora,cmd Markdown,Markdownpad2等
  • 收费编辑器: Mou,Ulysses,Byword,Marked2等
  • 优质在线编辑器:马克飞象,Markable,StackEdit等

Markdown 语法

标题

标题是每篇文章都需要也是最常用的格式，在 Markdown 中，如果一段文字被定义为标题，只要在这段文字前加 # 号即可.

强调标记

用*符号来标记文字为斜体或粗体，需要注意的就是，以什么开始，以什么结尾，数量也保持一致，相应的语法才能生效。

分割线

用*符合或-符号来生成分割线，重复三次即可。

列表

列表分无序列表合有序列表两类，前者以「*」、「-」、「+」开头，后者以阿拉伯数字「1」开头。

三个注意点：

  1. 如果前面有内容，在输入字符前，应先空一行；
  2. 符号与内容之间，须隔一个空格符，列表模式才能生效；
  3. 第一行内容输入完成，换行会自动补全下一行的开头符号（有序列表数字递增，无序列表符号不变），接下去只需继续输下一行内容，以此类推；
  4. 多层级列表，为美观也为减少出错考虑，注意同级列表符号的统一和上下对齐。

引用

使用「>」，添加在每行的开头。如果为嵌套引用，行多添加一个「>」符号，再输入文字内容即可（每行行尾记得添两个空格符）

两个注意点：

  1. 」的上一行，必须为空行；

  2. 「>」与其后紧跟的第一个字之间有无空格符不影响效果（不过建议加一个）。

代码区

连续三个反引号「`」组成的前后两行，将内容包裹起来。反引号的输入：英文输入模式下，点击键盘左上角的「~」键。

标记小段代码（文字）。为着重强调的内容添加深色背景框，在内容前后各添一个反引号「`」，将代码段或文字夹在中间即可实现。

网址链接

对于自动链接，使用「<」、「>」这样的尖角符号，url/email 在 Markdown 下可自动实现可点击链接的效果。网址的链接形式如下：

  • 一个方括号，添加图片的描述文字
  • 一个括号，添加图片网址

添加图片

相比插入网址链接，多了一个开头的惊叹号。在页面中，要并排插入多张图片（两张或三张并排显示）或将插入图片居中显示，又或者是类似固定显示图片的宽度与高度，目前依靠 Markdown 语法还无法实现。 这时，调用 HTML 语法就可以轻松搞定。图床在上传图片后会提供一个图片的外链。我们可以利用外链把图片添加到自己的博文中。图片也可以存储在 Github 自己的博客仓库里，不过图片蛮占用空间（Github 项目空间为200 M）。因而最好是选图床托管图片。

  1. 一个英文输入下的惊叹号「!」；
  2. 一个方括号，添加图片的描述文字；
  3. 一个括号，添加图片网址。

其他常见用法

任务列表

-加[ ],再加内容，或-加[ x ],再加内容，三个之间有空格符号。

文字中划线及标记

在文字中间划线，一对~~符号之间加文字即可。对于文字标记，一对==符号之间加文字。添加文字下划线，一对++符号之间加文字即可。
