第3天-学习python

发表于 2017-04-16   |   分类于 技术

引言

Anaconda是一个开源的数据科学平台，包含了100多种常用的Python数据科学包，默认集成了Notebook,也是非常好的一个包管理工具。完美的兼容python2.7和python3.5，免去配置环境变量的烦恼。Jupyter Notebook是一个运行在浏览器里面的web应用，可以非常方便的编辑、运行Python代码；添加文档（支持MarkDown）；演示运行结果等；还可以很方便的导出文件。完全可以取代在shell中写Python。Anaconda利用工具/命令conda来进行package和environment的管理，并且已经包含了Python和相关的配套工具

  • conda可以理解为一个工具，也是一个可执行命令，其核心功能是包管理与环境管理。包管理与pip的使用类似，环境管理则允许用户方便地安装不同版本的python并可以快速切换。Anaconda则是一个打包的集合，里面预装好了conda、某个版本的python、众多packages、科学计算工具等等，所以也称为Python的一种发行版。

Anaconda下载

  • 去官网下载Anaconda 下载地址，选择对应的你的系统版本。如果官网速度较慢，可以到这儿下载清华大学开源镜像中心

  • 选择你的安装路径，然后耐心等待，等到安装完成。

  • win健+R（或者点开始菜单-运行-输入cmd-按回车）打开命令提示符，输入conda --version查看版本号，如果可以查看到版本号则证明安装成功。直接通过conda命令管理python包。

  • Anaconda安装成功之后，我们可以修改其包管理镜像为国内源，在cmd中运行两行命令即可。
