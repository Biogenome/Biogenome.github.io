在cmd下运行Python脚本+如何使用Python Shell

原始的开发方式

  • 找个文本编辑器，新建个.py文件，写上Python代码
  • 打开Windows的cmd，并且切换到对应的python脚本所在目录
  • 在cmd中，输入你的Python脚本，即.py文件的完整的文件名，而接下来，作为正常的开发Python的流程，就是，你继续去修改你的Python代码，添加新的代码，然后保存python文件，再回到cmd中，重新再次运行Python，以验证程序运行是否正常，是否获得了你所期望的结果。一点点，由少到多，一点点写出足够复杂的Python代码，实现你的复杂的功能。

利用Python的shell进行交互式开发

  • Python有个shell，提供一个Python运行环境。方便你交互式开发。即写一行代码，就可以立刻被运行，然后方便查看到结果。在Windows下，分为命令行版本的Python Shell – Python (command line)和带图形界面的Python Shell – IDLE (Python GUI)，同样是Python的Shell，但是是相对于command line版本而言，是GUI，带图形界面的版本的。

利用第三方Python的IDE进行Python开发

  • IDE,集成开发环境,就是把开发相关的各种环境（和工具）都集成到一起，而Python IDE，把和Python开发相关的各种工具，即Python代码编辑器 + Python的运行环境：windows的cmd+有菜单和快捷键直接帮你实现运行对应的Python代码。比如sublime txt ,PyCharm等。

Sublime Text的使用

Sublime Text 是一款跨平台代码编辑器，在Linux、OS X和Windows下均可使用,并包含大量实用插件，我们可以通过安装自己领域的插件来成倍提高工作效率,有令行环境和图形界面环境，可以将两者结合起来使用。安装后之后可进行相应的设置，并进行安装相应的插件来满足实际工作的需求。
