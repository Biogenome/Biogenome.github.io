使用列表(list)和元组(tuple)

列表

列表是Python内置的一种数据类型，可以随时添加和删除其中的元素。

  1. 用len()函数可以获得list元素的个数.

  2. 用索引来访问list中每一个位置的元素，索引是从0开始.

  3. 如果要取最后一个元素，除了计算索引位置外，还可以用-1做索引，直接获取最后一个元素.以此类推，可以获取倒数第2个、倒数第3个,如[-2],[-3]等.

    1. list是一个可变的有序表，所以，可以往list中追加元素到末尾,例如classmates = ['Michael', 'Bob', 'Tracy']，输入命令classmates.append('Adam')，则 classmates=['Michael', 'Bob', 'Tracy', 'Adam'].也可以把元素插入到指定的位置，比如索引号为1的位置,classmates.insert(1, ‘Jack’).
    2. 要删除list末尾的元素，用pop()方法,例如classmates.pop(),要删除指定位置的元素，用pop(i)方法，其中i是索引位置.
    3. 要把某个元素替换成别的元素，可以直接赋值给对应的索引位置,如classmates[1] = 'Sarah'.
    4. list里面的元素的数据类型也可以不同,list元素也可以是另一个list.如果一个list中一个元素也没有，就是一个空的list，它的长度为0

    元组

    tuple和list非常类似，但是tuple一旦初始化就不能修改.它也没有append()，insert()这样的方法。其他获取元素的方法和list是一样的,因为tuple不可变，所以代码更安全。如果可能，能用tuple代替list就尽量用tuple.

    1. 要定义一个只有1个元素的tuple,为了防止和数学公式中的小括号产生歧义，规定只有1个元素的tuple定义时必须加一个逗号,，来消除歧义，例如t = (1,).

    条件判断

    计算机之所以能做很多自动化的任务，因为它可以自己做条件判断。在Python程序中，用if语句实现.也可以给if添加一个else语句，意思是，如果if判断是False，不要执行if的内容，去把else执行了.也可以用elif做更细致的判断。

    1. if语句是从上往下判断，如果在某个判断上是True，把该判断对应的语句执行后，就忽略掉剩下的elif和else.

    循环

    为了让计算机能计算成千上万次的重复运算，我们就需要循环语句,Python的循环有两种，一种是for…in循环，依次把list或tuple中的每个元素迭代出来,第二种循环是while循环，只要条件满足，就不断循环，条件不满足时退出循环.

    1. 在循环中，break语句可以提前退出循环.
    2. 在循环过程中，也可以通过continue语句，跳过当前的这次循环，直接开始下一次循环.

    使用dict和set

    • dict全称dictionary，在其他语言中也称为map，使用键-值（key-value）存储，具有极快的查找速度。
    1. 一个key只能对应一个value，所以，多次对一个key放入value，后面的值会把前面的值冲掉.要避免key不存在的错误，有两种办法，一是通过in判断key是否存在,二是通过dict提供的get方法，如果key不存在，可以返回None，或者自己指定的value.
    2. 要删除一个key，用pop(key)方法，对应的value也会从dict中删除.
    3. dict的key必须是不可变对象,这个通过key计算位置的算法称为哈希算法（Hash）,在Python中，字符串、整数等都是不可变的，因此，可以放心地作为key。而list是可变的，就不能作为key.
    • set和dict类似，也是一组key的集合，但不存储value。由于key不能重复，所以，在set中，没有重复的key,重复元素在set中自动被过滤
    1. 通过add(key)方法可以添加元素,通过remove(key)方法可以删除元素
    2. set可以看成数学意义上的无序和无重复元素的集合，因此，两个set可以做数学意义上的交集、并集等操作.
