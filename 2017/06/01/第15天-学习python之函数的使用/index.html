第15天-学习python之函数的使用

发表于 2017-06-01   |   分类于 技术

调用函数

Python内置了很多有用的函数，我们可以直接调用。要调用一个函数，需要知道函数的名称和参数，可以直接从Python的官方网站查看文档：python内置函数官方文档，也可以在交互式命令行通过help(abs)查看abs函数的帮助信息。

调用函数的时候，如果传入的参数数量不对或参数类型不能被函数所接受，会报TypeError的错误。

定义函数

在Python中，定义一个函数要使用def语句，依次写出函数名、括号、括号中的参数和冒号:，然后，在缩进块中编写函数体，函数的返回值用return语句返回。

return和print的区别

print：输出，或者翻译为打印，指输出方便人类来看的数值或文字，机器没有存储，也没法再次使用。如果机器想再次调用，则结果值为“none”。

return：返回值，表面上看也是输出了人类可以看的数值或文字，但其实核心是机器也可识别，可以存储在自己系统里，之后随时调用。

巧妙运用break和continue

break就是跳出/中止这个循环的意思。while True是一个常见的无限循环语句，如果没有break，这个循环体是会一直运行下去的，所以一定要小心！break的用法，要点如下：

  • 插入你的循环体中你想中止或跳出的地方。
  • 跳出循环体后，直接进入下一条语句。

相应地，continue也是跳出循环体，但不同的是，continue是直接回到循环体最初while的位置重新运行循环体，而break是到下一条语句。

这就是个continue和break的混合体，当你输入了#的时候，python会直接回到while True，从头再来，当你输入了break的时候，python会直接跳出循环体，到下一条语句，print ‘Done!’。

一句话总结，break和continue都是跳出循环体，但break往下走，continue往上走。

for…in…语句

与while循环句不同却类似，for…in…语句是用来查找已知序列中的某个值的语句，所以称为definite loop
