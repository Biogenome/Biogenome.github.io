第17天-学习python之常用模块

发表于 2017-06-15   |   分类于 技术

copy模块用来复制

copy模块用于对象的拷贝操作。该模块非常简单，只提供了两个主要的方法： copy.copy 与 copy.deepcopy ，分别表示浅复制与深复制。深复制表示原来列表中元素的改动不会影响新的列表，而浅复制会影响。

Kewword模块记录所有的关键字

iskeyword()表示字符串是否为关键字。kwlist表示列出所有python的关键字。

random模块获得随机数

random模块主要用来获得随机数，常用的函数为randint(),choice,还有shuffle.randit函数在一定数字范围内找出一个随机的数字，choice从列表中随机选出一个元素，shuffle用来给列表洗牌。

sys模块用来控制shell程序

exit函数停止控制台或输入程序。sys模块中的stdin对象，会提示用户输入信息，读取到shell程序中使用。它有一个readline函数，能读取从键盘输入的一行文本，直到用户按了回车键。它和input的区别在于可以指定一个参数来读取多少个字符。stdout,标准输出，用来写消息，与print类似。变量version表示python的版本。

pickle模块用来保存信息

pickle模块用来把python对象转换成可以方便写入到文件和从文件中读取的形式。

常见模块导入

模块也可称为库，可分为三大类：python标准库，第三方模块，应用程序自定义模块

python标准库：安装好python后，自带的库中的模块也叫做内置模块，直接import导入就可直接使用。从帮助文档中查找到这些模块。第三方模块：要安装后才能使用的模块。自定义模块：自己写的.py文件。以Tkinter模块为例简单介绍这几种模块导入的具体含义
